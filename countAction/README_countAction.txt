Cette procedure sert à connaitre combien de fois nous avons executé chaque action.
Elle peut révéler des choses interessantes!

Placer le programme Python countAction.py dans le même répértoire que le resultat de notre essai (.json). Executer la commande suivante:

./countAction.py XXXXXX.json | sort | uniq -c


permission non accordée: 
chmod -x ./countAction.py
